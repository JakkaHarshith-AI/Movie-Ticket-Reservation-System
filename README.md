# Movie Ticket Reservation System

## Overview
This project implements a Movie Ticket Reservation System using MySQL. It manages users, movie listings, seat availability, and booking records through relational database design.

## Features
- User registration
- Movie scheduling
- Ticket booking
- Seat availability tracking
- Relational database schema with foreign key constraints

## Technologies Used
- MySQL
- SQL

## Database Structure
- Users
- Movies
- Bookings

## How to Run
1. Import `database.sql` into MySQL.
2. Execute `queries.sql` to insert sample data and test booking operations.
